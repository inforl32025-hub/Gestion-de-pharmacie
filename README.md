# THEME : GESTION DE PHARMACIE

1-**DESCRIPTION DU PROJET**
Le projet **Gestion de Pharmacie** est une application web développée pour faciliter la gestion quotidienne d'une pharmacie.
Il permet de gérer efficacement les médicaments, les ventes, les fournisseurs, les commandes et les utilisateurs, tout en assurant une synchronisation en temps réel avec une base de données MySQL.

L'objectif principal est d'offrir un outil simple, fiable et professionnel permettant d'optimiser les opérations et de réduire les erreurs humaines.

2-**OBJECTIFS**
- Digitaliser la gestion d'une pharmacie
- Suivre les stocks en temps réel
- Faciliter la gestion des ventes
- Centraliser les informations
- Améliorer la traçabilité
- Réduire les erreurs manuelles
